EventEase Booking System

Overview
EventEase is an ASP.NET Core MVC web application designed to manage venues, events, and bookings efficiently. It helps prevent scheduling conflicts and ensures data integrity when managing related records.

Features
- Create, edit, and delete venues
- Create and manage events
- Book venues for events
- Prevent overlapping bookings
- Prevent deletion of venues with existing bookings
- Display user feedback using TempData alerts

Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

How to Run the Project
1. Open the project in Visual Studio
2. Ensure SQL Server is running
3. Run database migrations (if applicable)
4. Press F5 or click Run

Project Structure
- Controllers: Handles application logic
- Models: Represents data and database structure
- Views: User interface (Razor pages)
- wwwroot: Static files (CSS, JS)
